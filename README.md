# Number-Guessing-Game

## Project Description

This is a terminal based number guessing game written in Python.
The computer randomly generates a number between 1 and 100, and the player must guess the number based on the given feedback

After each guess, the program will indicate whether the guess is:
- Too high
- Too low
- Correct

The game will continue until the correct number is guessed.

---

## Features
- Random number generation
- Unlimited guesses
- Input validation
- Tracks number of attempts
- Stores and displays guess history
- User-friendly messages
